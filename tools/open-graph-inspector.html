⚠ Could not fetch the URL directly

This is usually caused by CORS restrictions — the target server doesn't allow browser-based requests from other origins.

How to fix it:

  1. Open the URL in a new browser tab.
  2. Right-click the page and choose View Page Source (or press Ctrl+U / Cmd+Option+U).
  3. Select all the HTML (Ctrl+A / Cmd+A) and copy it.
  4. Switch to Paste HTML mode and paste the source here.
